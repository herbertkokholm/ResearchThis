# docs/

- [`index.html`](index.html) — the public brochure/landing page, published via
  GitHub Pages at [herbertkokholm.github.io/ResearchThis](https://herbertkokholm.github.io/ResearchThis/).
  Static and self-contained (no build step, no fetch calls) — edit and push to
  `main` to redeploy. Styled to match the portal's own gallery UI; the three
  example finding cards on it link to real arXiv papers, not fabricated
  content.
- [`SPEC.md`](SPEC.md) — the portal's spec: data contract, server routes,
  refresh workflow, and the Zotero reconciliation design. Internal —
  deliberately not linked from the public brochure.

For running/deploying instructions, see the [repo root README](../README.md).
